# Expense Tracker

A full-stack expense tracking application built with Next.js 14, TypeScript, SQLite, and Tailwind CSS.

## Features

- ✅ View all expenses sorted by date (newest first)
- ✅ Filter expenses by category (Food, Transport, Utilities, Other)
- ✅ Add new expenses with validation
- ✅ Delete expenses with optimistic UI updates
- ✅ Server-side rendering for initial data load
- ✅ Real-time total spend calculation
- ✅ Responsive design with Tailwind CSS

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- SQLite with better-sqlite3 (raw SQL, no ORM)
- Tailwind CSS
